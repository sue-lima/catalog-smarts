## How to run

**Front-end**
```bash
# Enter the front-end folder with
$ cd smarts-challenge-reactjs/front

# Install the dependencies
$ npm install

# Run the application
$ npm start
```


**Back-end**
```bash
# Enter the back-end folder with
$ cd smarts-challenge-reactjs/back

# Install the dependencies
$ npm install

# Run the back-end
$ npm start
```
